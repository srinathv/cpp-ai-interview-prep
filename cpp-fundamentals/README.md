# C++ Fundamentals

Modern C++ features essential for AI/ML library development.

## Topics

1. **classes.cpp** - Classes, inheritance, polymorphism, operator overloading
2. **auto_and_decltype.cpp** - Type deduction
3. **lambdas.cpp** - Lambda expressions and captures
4. **move_semantics.cpp** - Move constructors and std::move
5. **smart_pointers.cpp** - unique_ptr, shared_ptr, weak_ptr
6. **templates.cpp** - Template basics and specialization
7. **ranges.cpp** - C++20 ranges library
8. **concepts.cpp** - C++20 concepts
9. **coroutines.cpp** - C++20 coroutines

## Key Interview Topics

- Class design and the Rule of Five
- Virtual functions and polymorphism
- When to use auto vs explicit types
- Lambda capture modes ([=], [&], [this])
- Move vs copy semantics
- Smart pointer ownership models
- Template metaprogramming basics
- Range-based algorithms
- C++20 concepts for template constraints
- Coroutines for async and lazy evaluation

## Build

```bash
g++ -std=c++20 -O2 <filename>.cpp -o <program>
```
