# C++ Fundamentals

Modern C++ features essential for AI/ML library development.

## Topics

1. **auto_and_decltype.cpp** - Type deduction
2. **lambdas.cpp** - Lambda expressions and captures
3. **move_semantics.cpp** - Move constructors and std::move
4. **smart_pointers.cpp** - unique_ptr, shared_ptr, weak_ptr
5. **templates.cpp** - Template basics and specialization
6. **ranges.cpp** - C++20 ranges library
7. **concepts.cpp** - C++20 concepts
8. **coroutines.cpp** - C++20 coroutines

## Key Interview Topics

- When to use auto vs explicit types
- Lambda capture modes ([=], [&], [this])
- Move vs copy semantics
- Smart pointer ownership models
- Template metaprogramming basics
- Range-based algorithms

## Build

```bash
g++ -std=c++20 -O2 <filename>.cpp -o <program>
```
