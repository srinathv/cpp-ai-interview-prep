# Data Structures

Essential data structures for technical interviews and AI/ML development.

## Topics

1. **vector_operations.cpp** - Dynamic arrays and operations
2. **hash_table.cpp** - Unordered maps and sets
3. **linked_list.cpp** - Singly and doubly linked lists
4. **binary_tree.cpp** - Tree traversals and operations
5. **graph.cpp** - Graph representations and algorithms
6. **custom_dataframe.cpp** - Simple DataFrame implementation

## Common Interview Questions

- Implement vector from scratch
- Hash table collision resolution
- Reverse a linked list
- Binary tree traversals (in/pre/post-order)
- Graph BFS/DFS

## Complexity Cheat Sheet

| Operation | Vector | Hash Map | Linked List | Binary Tree |
|-----------|--------|----------|-------------|-------------|
| Access    | O(1)   | -        | O(n)        | O(log n)*   |
| Search    | O(n)   | O(1)     | O(n)        | O(log n)*   |
| Insert    | O(n)   | O(1)     | O(1)        | O(log n)*   |
| Delete    | O(n)   | O(1)     | O(1)        | O(log n)*   |

*Assuming balanced tree
